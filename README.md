
#  Movie Recommendation System

This project implements a simple **Machine Learning based Movie Recommendation System** using TF-IDF vectorization and cosine similarity.

##  Project Description

The system analyzes movie genres and calculates similarity between movies.
It recommends movies based on similarity scores.

##  Algorithm Used

* TF-IDF (Term Frequency–Inverse Document Frequency)
* Cosine Similarity

## Technologies Used

* Python
* Pandas
* Scikit-learn

##  Project Files

movie_recommender.py → main recommendation algorithm
movies.csv → sample dataset of movies and genres
